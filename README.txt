
BetterDeal – lauffähige Minimal-Seite (PHP, Bootstrap 5, FA5)
-------------------------------------------------------------

Struktur
- index.php                 Startseite mit Hero, Warum, Ablauf, Kalkulation, Makler‑Block
- includes/header.php       <head>, Navigation, CSS/Icons (FA5), Button öffnet Modal
- includes/footer.php       Schwarzer Footer + Bootstrap Bundle + app.js
- assets/css/styles.css     Styles inkl. Icon‑Fix + Footer
- assets/js/app.js          Kleines Script

Rechtliches (Platzhalter)
- impressum.php, datenschutz.php, agb.php

Installation
1) ZIP auf den Server kopieren und entpacken.
2) Domain/Subfolder auf index.php zeigen lassen.
3) Falls Sie ein Logo als Bild nutzen möchten, legen Sie es in assets/img/ ab und tauschen Sie das Brand‑Element aus.

Hinweise
- Font Awesome 5 und Bootstrap Icons werden per CDN geladen.
- Der Button „Meinen Verkaufspreis ermitteln“ öffnet ein funktionierendes Bootstrap‑Modal (#preisrechnerModal).
- Die drei Kacheln verwenden echte Icons (fa-shield-alt, fa-home, fa-chart-line).
